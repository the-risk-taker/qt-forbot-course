# Qt Course for Forbot

The source code I made for all examples and programs in the [Qt course on Forbot](https://forbot.pl/blog/kurs-qt-1-czym-jest-qt-pierwsza-aplikacja-w-praktyce-id35549) [written in Polish].

Parts:

1. About Qt, first program
2. Serial communication (UART) between desktop na Arduino dev board
3. First mobile application in Qt for Android
4. Bluetooth communication between smartphone na Arduino dev board with Bluetooth Classic module
5. Building interfaces with Qt Quick and QML
6. Passing data between C++ <-> QML
7. Intro to concurrent processing in Qt
8. Intro to multithreading in Qt
9. Intro to multitasking in Qt

[Forbot](https://forbot.pl/blog/) is the most popular blog in Poland, regularly publishing guides for electronics and DIY enthusiasts who are passionate about building electronic devices on their own.

## Setting up, building and running examples

The easiest way to setup, build and run examples is to open root `CMakeLists.txt` in QtCreator.
